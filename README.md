💰 Tax Calculator (Core Java)

📌 Description

This is a simple Tax Calculator application developed using Core Java.
It calculates income tax based on the salary entered by the user using predefined tax slabs.

🚀 Features
Takes salary input from the user
Calculates tax based on different income ranges
Uses multiple tax rates (0%, 5%, 10%, 15%, 20%, 30%)
Displays the calculated tax amount
Simple and easy-to-use console-based application

🛠️ Technologies Used
Core Java
Scanner Class (for user input)

⚙️ How It Works
User enters their salary
Program checks salary range using if-else conditions
Applies the appropriate tax percentage
Displays the total tax amount

📊 Tax Slabs Used
Below ₹3,00,000 → No Tax
₹3,00,000 – ₹7,00,000 → 5%
₹7,00,000 – ₹10,00,000 → 10%
₹10,00,000 – ₹12,00,000 → 15%
₹12,00,000 – ₹15,00,000 → 20%
Above ₹15,00,000 → 30%

🎯 Learning Outcome
Understanding of conditional statements (if-else)
Handling user input in Java
Applying logic to solve real-world problems
